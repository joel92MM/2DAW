5.- TIME

El objetivo de esta actividad es calcular el tiempo que tarda un script a ejecutarse. Para ello, utilizaremos la función Date (), que tiene precisión de milisegundos. En Calculating PI to an Arbitrary number of digits tiene un algoritmo para encontrar los decimales de PI que es fácil de reutilizar en su código. Solo tendrá que calcular los primeros 60 números de PI (cosa bastante evidente a partir del código disponible), y ver qué tarda en calcularlos. Debes crear una tabla para ver cuanto tarda en calcular desde 1 a 60 dígitos, es decir cuánto tarda en calcular el primero, el segundo, etc. y así iremos viendo que cada vez le cuesta más (aunque el algoritmo es muy rápido).
